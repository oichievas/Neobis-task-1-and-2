Задача #1

                Задача #1
    for (var i = 0; i < 10; i++) {
        setTimeout(function() {
            console.log(i);
        }
    };
    Что выведет консоль и почему?

Выведет синтаксическую ошибку. Так как в функции с setTimeout нужно задать время. Функция должна выглядеть так: 
                    for (var i = 0; i < 10; i++) {
                        setTimeout(function() {
                            console.log(i);
                        }, 2000)
                    };
